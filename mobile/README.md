# FocusStudy AI Mobile

Aplicacion Flutter para Android.

## Responsabilidades

- UI y navegacion.
- Estado con Riverpod.
- Comunicacion con backend via Dio.
- Auth y storage cliente con Supabase.
- Integracion Android profunda via Platform Channels y Kotlin.

## Comandos

```powershell
flutter pub get
flutter analyze
flutter test
```

Para emulador Android, usar el AVD `FocusStudy_API36`.
